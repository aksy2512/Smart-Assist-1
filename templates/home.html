About

Smart Assist is a Machine Learning based web application which helps students by providing some useful insights from long video, audio and text files. This websitess also provides mock test based on the given lecture text files. Finding important points and keywords from the paragraph. This application also provides various maathematical and scientific tools to assist the students. You must have encountered the problem of studying long chapters which are quite boring to read this apps provides a short summary of a whole chapter which can be useful at the time of revision as well.
